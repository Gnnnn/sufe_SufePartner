此为校园互助平台系统。使用技术：三剑客+php+mysql。
系统分为两个模块。
模块一：互助，主要逻辑为发布--接受/删除--完成/删除。
模块二：问答，相比一较为简单，主要逻辑为发布问题--回答问题。
模块三：用户，login&register&userinfor
